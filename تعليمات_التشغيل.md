# تعليمات إعداد وتشغيل المشروع

## الخطوات السريعة

### 1️⃣ إنشاء قاعدة البيانات

افتح **pgAdmin** أو **psql** وأنشئ قاعدة البيانات:

```sql
CREATE DATABASE dtc_job_bot;
```

### 2️⃣ إعداد البيئة الافتراضية

```bash
# إنشاء البيئة
python -m venv venv

# تفعيلها (Windows)
venv\Scripts\activate

# تفعيلها (Linux/Mac)
source venv/bin/activate
```

### 3️⃣ تثبيت المكتبات

```bash
pip install -r requirements.txt
```

### 4️⃣ إعداد ملف .env

أنشئ ملف `.env` في المجلد الرئيسي:

```env
BOT_TOKEN=رمز_البوت_من_BotFather
DB_HOST=localhost
DB_PORT=5432
DB_NAME=dtc_job_bot
DB_USER=postgres
DB_PASSWORD=كلمة_مرور_postgres
```

### 5️⃣ إنشاء الجداول (اختياري)

```bash
python setup_database.py
```

### 6️⃣ تشغيل البوت

```bash
python main.py
```

---

## ملاحظات مهمة

✅ تأكد من تشغيل PostgreSQL قبل تشغيل البوت
✅ تأكد من صحة إعدادات قاعدة البيانات في `.env`
✅ البوت سينشئ الجداول تلقائياً عند التشغيل الأول

---

## استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| لا يمكن الاتصال بقاعدة البيانات | تأكد من تشغيل PostgreSQL |
| قاعدة البيانات غير موجودة | أنشئها باستخدام `CREATE DATABASE dtc_job_bot;` |
| خطأ في كلمة المرور | تحقق من `DB_PASSWORD` في `.env` |
| خطأ في BOT_TOKEN | احصل على رمز من [@BotFather](https://t.me/botfather) |
